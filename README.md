# create-avif

A CLI tool that converts images (`jpg`, `png`, `svg`) to `avif` format images.

[![npm](https://img.shields.io/npm/v/create-avif?logo=npm)](https://www.npmjs.com/package/create-avif)

## usage

`pnpm create avif`
